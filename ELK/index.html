GGK Infrastructure Monitoring
About Monitoring

GGK Infrastructure Monitoring includes:

  • Filrewalls (Fortigate, Cisco ASA)
  • Switches
  • Service providers (Airtel, Beam, Beam_1gig, Pioneer, TATA, UPPAL LAN) bandwidts
  • UPS (Uninterruptable Power Supply)
  • NVR (Network Video Recorder), DVR (Digital Video Recorder)
  • Active directory event logs
  • All devices logs with rsyslog server
